Ihre Anfrage wurde seit mehr als drei Monaten nicht beantwortet? Möglicherweise können Sie eine Untätigkeitsklage einreichen. Mit unserem Klageautomaten können Sie überprüfen, ob eine Klage in Betracht kommt. Wenn das der Fall ist, wird direkt eine Klageschrift für Sie erstellt.

Voraussetzung hierfür ist zunächst, dass die Behörde innerhalb von 3 Monaten und ohne zureichenden Grund nicht reagiert hat. Die Klage kann dann, anders als sonst, erhoben werden, ohne dass zuvor ein Widerspruchsverfahren abgeschlossen worden ist (§ 75 VwGO).

Der Klageautomat wird Schritt für Schritt alle relvanten Informationen abfragen und, wenn alle Voraussetzungen erfüllt sind, eine Klageschrift generieren.
